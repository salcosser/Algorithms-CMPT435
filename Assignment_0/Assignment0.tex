\section{ A Couple of Limericks About Java}

\noindent
Tom moved to Java from JavaScript\\
And at that moment he was gripped.\\
    \hspace*{1.5em}While memorizing public static void main,\\
    \hspace*{1.5em}Will temporarily be his bane,\\
 By knowing a more general purpose language he is now better equipped.
\vspace{2em}


\noindent
Tom has now been using Java for a bit\\
And there's a couple of things in the docs that don't quite fit.\\
    \hspace*{1.5em}With StringBuilder and StringBuffer,\\
    \hspace*{1.5em}He is adamant that Oracle wants him to suffer,\\
But never for a minute did Tom ever think to quit.
